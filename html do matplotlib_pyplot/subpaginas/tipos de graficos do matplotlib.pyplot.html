Escolha do Tipo de Gráfico

Os gráficos são ferramentas essenciais para visualizar e interpretar dados. O Pyplot oferece diversas opções para criar gráficos que atendem a diferentes necessidades. Aqui estão os tipos mais utilizados, suas características e aplicações:

Gráfico de Linha (plt.plot):

O gráfico de linha é amplamente utilizado para mostrar a relação entre variáveis ao longo de um intervalo contínuo. Ele conecta os pontos de dados com linhas, ajudando a visualizar tendências e mudanças ao longo do tempo.

Características:

  • Indicado para dados sequenciais ou temporais.
  • Destaca variações suaves e contínuas.
  • Facilita a comparação entre múltiplas séries de dados em um único gráfico.

Quando usar:

  • Ideal para visualizar séries temporais, como flutuações de preço, crescimento de audiência, ou evolução de métricas.
  • Também é útil para monitorar tendências, como o desempenho de uma empresa ou progresso acadêmico.

Exemplo: média de usuários ao longo do ano.


meses = ['Jan', 'Fev', 'Mar', 'Abr', 'Mai', 'Jun', 'Jul', 'Ago', 'Set', 'Out', 'Nov', 'Dez']
novos_usuarios = [100, 80, 110, 130, 130, 160, 170, 120, 110, 150, 200, 140]

plt.plot(meses, novos_usuarios)  # Cria uma linha conectando os pontos (1, 4), (2, 5) e (3, 6)

plt.title("média de usuários ao longo do ano")
plt.ylabel("usuários (em milhar)")

plt.show()

Por que usar: Ele oferece uma maneira intuitiva de interpretar dados contínuos, destacando mudanças de forma clara.

Gráfico de Dispersão (plt.scatter):

Este tipo de gráfico utiliza pontos no plano cartesiano para representar pares de valores, sendo excelente para identificar correlações e padrões.

Características:

  • Mostra a relação entre duas variáveis quantitativas.
  • Útil para detectar agrupamentos, tendências ou outliers nos dados.
  • Permite personalização, como variação no tamanho ou cor dos pontos para representar dimensões adicionais.

Quando usar:

  • Para explorar a relação entre fatores, como tempo de prática esportiva e desempenho físico, ou entre preço e demanda de um produto.

Exemplo: nota e tempo de estudo de 20 alunos.


tempo_estudo = [1, 3.5, 1, 3, 2.5, 7, 1, 3, 3, 5.3, 7, 6, 4, 4.3, 3, 6, 6, 5, 5, 2, 4.7]
notas = [3, 7, 2, 4, 5, 9, 5, 8, 5, 9, 10, 8, 7, 6, 8, 10, 8, 10, 9, 6, 7.2]

plt.scatter(tempo_estudo, notas)  # Plota pontos individuais para cada par de (x, y)

plt.title("Notas dos Alunos")
plt.xlabel("Horas")

plt.show()

Por que usar: Proporciona uma visão clara de como uma variável influencia outra, ajudando a responder perguntas como: "Quanto mais uma pessoa estuda, melhores são suas notas?"

Gráfico de Barras (plt.bar):

O gráfico de barras é uma das formas mais comuns de visualização para comparar valores de diferentes categorias.

Características:

  • As barras verticais ou horizontais são proporcionais aos valores representados.
  • Permite comparar diretamente várias categorias de forma visualmente impactante
  • Suporte para gráficos agrupados ou empilhados para análises mais detalhadas.

Quando usar:

  • Perfeito para visualizar vendas mensais, desempenho por equipe ou preferência por diferentes produtos ou serviços.

Exemplo: Comparar a receita mensal de uma loja ao longo dos anos.

 
receita_mensal = [10000, 7000, 9800, 20000, 17000]
meses = ["2021", "2022", "2023", "2024", "2025"]

plt.bar(meses, receita_mensal)  # Plota barras com alturas baseadas nos valores

plt.title("receita mensal")
plt.ylabel("média da receita")

plt.show()

Por que usar: Ajuda a identificar rapidamente as diferenças e semelhanças entre categorias.

Histograma (plt.hist(dados, bins=n)):

Representa a distribuição de frequência de um conjunto de dados. Divide os dados em intervalos (bins) e mostra quantos valores estão em cada intervalo.

Características:

  • Mostra a frequência com que os dados caem dentro de certos intervalos.
  • Permite detectar padrões como dispersão, centralidade e outliers.
  • Flexível na escolha do número de bins, o que pode impactar a visualização.

Quando usar:

  • Ideal para análises de dados contínuos, como distribuição de idades, altura, peso ou notas.

Exemplo:


  plt.hist([1, 2, 2, 3, 3, 3, 4, 5, 5], bins=5)  # Agrupa os valores em 5 intervalos
  plt.show()

Por que usar: Ele destaca de forma clara onde os dados estão concentrados, permitindo análises detalhadas da variabilidade dos dados.

Gráfico de Pizza (plt.pie(valores)):

Mostra proporções ou partes de um todo, útil para dados categóricos em percentual.

Características:

  • Fácil de interpretar visualmente quando há poucas categorias.
  • Útil para mostrar divisões de recursos ou participações de mercado.
  • Pode incluir rótulos e destaques (como destacar a maior fatia).

Quando usar:

  • Ideal para mostrar como um total é distribuído entre várias categorias, como orçamentos, divisão de despesas ou participação de votos.

Exemplo:divisão do orçamento por protudos.


  quantidades_vendidas = [45, 30, 15, 10]
  nomes = ["batata", "batata frita", "pizza", "queijo"]

  plt.pie(quantidades_vendidas, labels=nomes)
  plt.title("divisão do orçamento por setores")

  plt.show()

Por que usar: Ele comunica de forma intuitiva e rápida como diferentes partes contribuem para o todo.
