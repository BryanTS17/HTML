todos formados Aceitos pelo matplotlib.pyplot

    1. Formatos de Dados Aceitos

    O Pyplot é muito flexível e aceita diversos formatos de dados para os eixos x e y:



  • Listas Python (list):

    Boa compatibilidade: São suportadas nativamente pelo Pyplot, sem a necessidade de bibliotecas adicionais.

    Flexíveis: Permitem a inclusão de diferentes tipos de dados, embora o Pyplot geralmente exija valores numéricos.

    Quando usar:

    • Quando você tem dados pequenos e já organizados.
    • Para experimentos rápidos ou prototipagem.
    • Para gráficos simples e diretos, como linhas ou dispersões.

    Exemplo de aplicação:

    Criar um gráfico básico para um conjunto pequeno de valores

    Python

        x = [1, 2, 3, 4, 5]
    y = [1, 4, 9, 16, 25]

  • Tuplas Python (tuple):

    Imutabilidade: Tuplas garantem que os dados não serão alterados acidentalmente.

    Eficiência:Consomem menos memória que listas, o que pode ser vantajoso em aplicações que exigem desempenho.

    Quando usar:

    • Quando os dados são estáticos e não precisam ser alterados durante a execução.
    • Para conjuntos de dados pequenos que você deseja proteger contra modificações.

    Exemplo de aplicação:

    Gráficos em que os valores são fixos, como dados pré-determinados de uma função:

    Python

        x = (1, 2, 3, 4, 5)
    y = (1, 4, 9, 16, 25)

  • Arrays NumPy (numpy.array):

    operações vetorizadas: Suportam operações matemáticas vetorizadas (cálculos em todos os elementos de uma vez).

    dimenções: Podem ser unidimensionais (1D) ou multidimensionais (2D, 3D, etc.).

    operações Avançadas: um array do numpy tem suporte para calculos científicos, Operações Matemáticas Avançadas, Álgebra Linear, Suporte a Grandes Volumes de Dados e muito mais.

    Quando usar:

    • Quando os dados são grandes e requerem manipulação matemática avançada.
    • Quando você precisa de desempenho ao processar ou manipular os dados..

    Exemplo de aplicação:

    Python

    
    import matplotlib.pyplot as plt
    import numpy as np

    x = np.array([1, 2, 3, 4, 5])
    y = x ** 2  # Eleva cada elemento ao quadrado

    plt.plot(x, y)
    plt.title("Exemplo com Arrays NumPy")
    plt.show()

  _

  • DataFrames do pandas (pandas.DataFrame): Quando os dados vêm de uma tabela, é comum usar pandas para manipulação e seleção

    Descrição:

    • Um DataFrame é uma estrutura tabular do Pandas, semelhante a uma tabela de banco de dados ou planilha Excel.
    • Cada coluna de um DataFrame pode ser interpretada como uma lista ou array.

    Quando usar:

    • Quando os dados são estruturados em tabelas (ex.: linhas e colunas).
    • Para visualização de tendências ou comparações entre categorias (ex.: vendas por ano, crescimento populacional).
    • Quando é necessário limpar ou transformar os dados antes de plotar.

    Exemplo de aplicação:

    Python

        import pandas as pd
    import matplotlib.pyplot as plt

    # Criando o DataFrame

    df = pd.DataFrame({
    'Ano': [2018, 2019, 2020, 2021],
    'Vendas': [150, 200, 250, 300]
    })

    # Extraindo dados para os eixos
    x = df['Ano']
    y = df['Vendas']

    plt.plot(x, y, marker='o')
    plt.title("Exemplo com DataFrame do Pandas")
    plt.show()

  • Valores Calculados: Você pode calcular os valores diretamente

    Descrição:Em muitos casos, os dados do gráfico não vêm de uma fonte externa, mas são calculados dinamicamente usando expressões matemáticas ou loops.

    dados dinamicos: Permite gerar dados no momento da execução.

    gráficos matemáticos:Útil para gráficos que representam funções matemáticas ou padrões.

    Quando usar:

    • Quando os dados dependem de uma regra matemática ou precisam ser gerados no momento do uso.
    • Para gráficos de funções ou simulações.

    Exemplo de aplicação:

    GPlotar gráficos de funções matemáticas:

    Python

        import matplotlib.pyplot as plt

    x = range(0, 10)  # Valores de 0 a 9
    y = [i**2 for i in x]  # Quadrado de cada valor

    plt.plot(x, y)
    plt.title("Exemplo com Valores Calculados")
    plt.show()
