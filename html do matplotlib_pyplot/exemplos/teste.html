Guia do matplotlib.pyplot

Uma introdução completa à criação de gráficos em Python

O que é matplotlib.pyplot?

O matplotlib.pyplot é uma biblioteca essencial para visualização de dados em Python. Ele permite criar gráficos de forma simples e personalizável.

Recursos Principais

  • Criação de gráficos de linha, barras, dispersão e mais
  • Personalização completa dos elementos gráficos
  • Integração com pandas e numpy
  • Possibilidade de criar gráficos 3D

Exemplo de Código


import matplotlib.pyplot as plt

x = [1, 2, 3, 4]
y = [10, 20, 25, 30]
plt.plot(x, y, color='#f56e06', marker='o')
plt.title("Gráfico de Exemplo")
plt.show()
